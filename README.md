# Merge-Sort
merge sort patika exercise

- write sorting phases of the following array [16,21,11,8,12,22] :

			[16,21,11]					[8,12,22] 
		[16]		[21,11]				[8]		[12,22]
		[16]	[21]	[11]				[8]	[12]	[22]             
			[16,21]	[11]				[8,12]		[22]
			[11,16,21]					[8,12,22]
					[8,11,12,16,21,22]
          
 - write big-O notation case :
    
    O(nlogn)
